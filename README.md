# Refund Request System

Sistema de gerenciamento de solicitações de reembolso desenvolvido em JavaScript.

O projeto simula o fluxo de registro e processamento de pedidos de reembolso, aplicando regras de negócio para validar solicitações e organizar o controle das requisições.

Este tipo de sistema é comum em plataformas de pagamento, e-commerce e serviços online.

---

## Objetivo do projeto

O objetivo do projeto é praticar o desenvolvimento de aplicações com lógica de negócio, simulando um sistema capaz de registrar e processar solicitações de reembolso.

A aplicação permite gerenciar pedidos enviados por usuários e aplicar regras para validar e organizar essas solicitações.

---

## Tecnologias utilizadas

- JavaScript
- HTML
- CSS

---

## Funcionalidades

O sistema permite:

- registrar solicitações de reembolso
- visualizar pedidos cadastrados
- organizar informações das solicitações
- aplicar regras básicas de processamento

---

## Estrutura do projeto

O projeto está organizado da seguinte forma:

```
src/
  modules/
  services/
  utils/
```

Essa estrutura ajuda a separar responsabilidades dentro da aplicação, deixando o código mais organizado e fácil de manter.

---

## Funcionamento do sistema

O fluxo básico do sistema é:

1. o usuário envia uma solicitação de reembolso
2. o sistema registra os dados do pedido
3. as informações são processadas e organizadas
4. o sistema exibe o histórico de solicitações

---

## Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/DanAntunesDev/refund-system
```

Acesse a pasta do projeto:

```bash
cd refund-system
```

Abra o arquivo principal no navegador ou execute o ambiente de desenvolvimento conforme configurado no projeto.
